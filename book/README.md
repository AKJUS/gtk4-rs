# The GTK-Rust Book

At the moment, it is deployed [here](https://hofer-julian.pages.gitlab.gnome.org/gtk-rs-book).


## Build instructions

1. Install mdbook

```
cargo install mdbook
```

2. Building

To view the book, type:

```bash
$ mdbook serve
```


To run the tests:

```bash
$ mdbook test
```
