# The GTK-Rust Book

At the moment, it is deployed [here](https://hofer-julian.pages.gitlab.gnome.org/gtk-rs-book).


## Build instructions

1. Install mdbook

```bash
$ cargo install mdbook
```

2. Building

To view the book, type:

```bash
$ mdbook serve
```

## Tests

To run the tests:

```bash
$ cd tests
$ cargo test
```
